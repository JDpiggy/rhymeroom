# RhymeRoom – Free Online Rap Lyric Writing Studio

RhymeRoom is a simple web app for rappers and songwriters to write, edit, and test lyrics while listening to beats in real time. It runs in the browser, saves your work locally, and is meant to feel like a focused writing notebook.

## Features

### Core lyric editor

- Notebook-style editor with a clean, distraction-free layout  
- Optional line numbers you can toggle on or off  
- Auto-save to browser local storage (saves roughly every second)  
- Tab key support for indentation  
- Multiple projects: create, rename, and delete lyric projects  
- Included sample lyrics so new users can see how it works  

### Rhyme detection and analysis

- Real-time rhyme checking with four comparison modes:  
  - Last word  
  - Last 2 words  
  - Last syllable  
  - Last 2 syllables  

## Getting started

1. Open the app in your browser.
2. Create a new project or load the default example.
3. Start typing your lyrics in the editor while playing your beat.
4. Use the rhyme mode options to see what rhymes and where.

## Notes

- All data is stored in your browser’s local storage.
- Clearing browser data or switching devices will remove your projects unless you back them up manually.
